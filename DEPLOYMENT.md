## Project Overview
CedarWire is a premium lifestyle publication (Refinery29 meets Vox aesthetic) covering 7 categories across 15 in-depth articles.

### Design Implementation
- **Colors**: Warm white (#FFFBF5) body, sage green (#5B8C5A) accent, terracotta (#C57B57) highlight, charcoal (#2C2C2C) text
- **Fonts**: DM Serif Display (headlines), Work Sans (body), Inter (UI)
- **Aesthetic**: Modern lifestyle editorial — warm, sophisticated, readable

### Technical Stack
- React 18 + TypeScript + Vite
- Tailwind CSS with custom theme
- React Router for SPA navigation
- React Helmet Async for SEO
